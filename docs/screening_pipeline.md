# 选股流程说明

## 整体架构

```
全量 A 股 (~5000只)
    │
    ▼ [阶段一：快照预筛] ─ 一次批量请求，毫秒级
    │
    ├── 策略一候选池 (~100-300只)
    │        放量活跃股：价格≥5元 + 成交额≥1亿 + 换手率≥2% + 量比≥1
    │
    └── 策略二候选池 (~50-200只)
             有明显下跌的股票：涨跌幅在 [-10%, -1%] 之间
    │
    ▼ [阶段二：逐只精细评分] ─ 对候选池逐只调接口
    │
    ├── 策略一：18维评分，得分≥7 进报告
    └── 策略二：6维评分，得分≥3 进报告
    │
    ▼ 生成 reports/screening/daily_screen_YYYYMMDD.md
```

---

## 阶段一：全市场快照预筛

### 数据来源

调用 `akshare.stock_zh_a_spot_em()`，**一次请求**拿到全市场所有 A 股的实时行情：

| 字段 | 说明 |
|------|------|
| 代码 | 6位股票代码 |
| 名称 | 股票简称 |
| 最新价 | 当前价格 |
| 涨跌幅 | 当日涨跌幅 (%) |
| 成交额 | 当日成交金额 (元) |
| 换手率 | 当日换手率 (%) |
| 量比 | 当日量比（当前成交量/5日均量） |

> **注意**：该接口为实时行情，仅在**交易时段（9:15-15:30）**有效。

### 通用过滤（两个策略都执行）

- 排除**北交所**股票（代码以 `8` 开头）
- 排除 **ST / 退市**股票（名称含 ST 或"退"）

### 策略一候选池筛选条件

目标：筛出**放量活跃、有上涨动力**的股票

| 条件 | 阈值 | 目的 |
|------|------|------|
| 最新价 | ≥ 5 元 | 过滤仙股，流动性差 |
| 成交额 | ≥ 1 亿 | 过滤僵尸股，确保有足够资金参与 |
| 换手率 | ≥ 2% | 有活跃交投 |
| 量比 | ≥ 1 | 今日成交量高于5日均量，放量特征 |
| 涨跌幅 | ≥ -8% | 排除接近跌停的股票 |

预期结果：5000 → **100~300 只**

### 策略二候选池筛选条件

目标：筛出**有明显下跌、值得关注抄底机会**的股票

| 条件 | 阈值 | 目的 |
|------|------|------|
| 最新价 | ≥ 3 元 | 过滤极低价股 |
| 成交额 | ≥ 5000 万 | 确保基本流动性 |
| 涨跌幅 | < -1% | 今日有明显下跌 |
| 涨跌幅 | > -10% | 排除跌停股（跌停通常是利空，不是抄底机会） |

预期结果：5000 → **50~200 只**

## 阶段二：逐只精细评分

对阶段一筛出的候选池，逐只调取更细粒度的数据（日线 K 线、资金流向、板块信息等），按策略维度打分。

### 策略一：强势多头突破（18维评分）

适用场景：市场情绪较好，做**强势股和龙头股**

**市场情绪前置门**（任一触发则跳过策略一全部股票）：
- 大盘 KDJ 极弱（K<20 且 J<0）→ 发出警告
- 涨跌家数比极差（跌家数 > 涨家数 2 倍以上）→ 直接跳过

**股票内置快速预筛**（策略一内部的二次预筛）：
- 价格在 MA20 的 97% 以上 OR 量比 > 0.8 → 进入完整评分
- 否则跳过，减少接口调用

**18 维评分明细**：

| 组别 | 维度 | 说明 |
|------|------|------|
| A. 前置过滤 | 基本面-PE | 市盈率合理区间 |
| | 基本面-业绩增长 | 净利润连续增长 |
| | 板块前五 | 所在板块在今日涨幅前5 |
| | 板块涨停 | 所在板块有涨停股带动 |
| B. 趋势条件 | 均线多头排列 | MA5 > MA10 > MA20 |
| | 缠论底分型 | 日线出现底分型结构 |
| | 头肩底形态 | 近期K线构成头肩底 |
| C. 动能条件 | MACD金叉>MA20 | MACD金叉且在MA20以上 |
| | KDJ>50 | KDJ处于多头区间 |
| | DMI手拉手 | ADX上扬，PDI > MDI |
| D. 主力量价 | 博弈长阳 | 近期出现大阳线（涨幅>5%） |
| | 量能放大 | 近日成交量明显放大 |
| | 九五之尊 | 收盘价接近当日最高价（95%以上） |
| E. 日内触发 | 高开 | 今日高开（相对昨收） |
| | 强分时 | 分时走势强于大盘 |
| | 量比>1 | 当日量比 > 1 |
| | 换手率>3% | 当日换手率 > 3% |
| | 资金流入 | 大单净流入为正 |

**进报告门槛**：得分 ≥ 7/18

---

### 策略二：缠论深度抄底（6维评分）

适用场景：标的经历大幅下跌，寻找**大级别转折点**

**股票内置快速预筛**：
- 近 60 日跌幅 > 10% → 进入完整评分
- 否则跳过（跌得不够，不值得抄底分析）

**6 维评分明细**：

| 组别 | 维度 | 说明 |
|------|------|------|
| A. 大级别背离 | 日线MACD底背离 | 价格创新低但MACD未创新低 |
| | 周线MACD底背离 | 周线级别的底背离（共振更强） |
| B. 极度超卖 | CYS < -15 | 筹码超跌指标极度超卖 |
| | CD40 < -20 | 40日动量指标极度超跌 |
| C. 底分型确认 | 底分型确认 | 日线出现底分型（不见底分型不抄底） |
| | 资金流入 | 抄底后开始有资金介入 |

**进报告门槛**：得分 ≥ 3/6

---

## 报告格式

报告保存至 `reports/screening/daily_screen_YYYYMMDD.md`，格式示例：

```
## 策略一：强势多头突破

- ✅ 002384 东山精密  9/18分 | 均线多头排列 | 缠论底分型 | KDJ>50 | ...
- ⚠️ 605258 协和电子  8/18分 | 均线多头排列 | 缠论底分型 | ...

## 策略二：缠论深度抄底

- ✅ 688256 寒武纪-U  3/6分 | CD40<-20(动量超跌) | 底分型确认 | 资金流入
```

评分 Emoji 含义：

| Emoji | 得分比例 |
|-------|---------|
| 🔥 | ≥ 70% |
| ✅ | ≥ 50% |
| ⚠️ | ≥ 30% |
| ❌ | < 30% |

## 使用方法

```bash
# 全市场两阶段扫描（推荐，需在交易时段运行）
python main.py --screen-all --no-market-review --no-notify

# 仅对自选股评分（任意时间可运行，使用 .env 中的 STOCK_LIST）
python main.py --no-market-review --no-notify

# 完整流程（选股 + LLM个股分析 + 大盘复盘）
python main.py --screen-all
```

> `--screen-all` 需在**交易时段（09:15-15:30）**运行，因为阶段一依赖实时行情数据。
